# SysPeek

SysPeek is a system monitor indicator that displays CPU usage,memory usage,
swap usage, disk usage and network traffic.

## Installation

1. Install dependencies.

        sudo apt install python3-setuptools

3. Install SysPeek.

        ./setup.py install

3. Run SysPeek.

        syspeek
